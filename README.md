## üìå Overview

ElectroStore is a fully functional web application built using **Python**, **Django**, **HTML**, and **CSS**.  
The app simulates an online electronics store where users can:

- Browse products  
- View detailed descriptions  
- Add products to a shopping cart  
- Increase quantities by clicking multiple times  
- Remove a single unit  
- Remove all units of a product  
- View totals and subtotals  

This project fulfills **all required and unique requirements** for the **Web Apps Module**.
